# Test Data

These Atari DEGAS image files are from the TOSEC (The Old School Emulation
Center) preservation collection and are used here for format compatibility
testing only. No claim of copyright is made over these files.

## Sources

- **MOUSE.PI1** — From "Mouse Trap" game, LSD Compacted Disk #01
- **AMMO.PC1** — From Falcon Compilation (NOMAD), armament screen artwork
- **VALENTIN.PI2** — Valentine clipart, Atari ST Review Issue 10 coverdisk
- **HIDDEN.PI3** — Hidden screen from "Democrazy" demo (The Germs, 1993)
- **CLIPART.PC3** — Atari computer clipart, Atari ST User Issue 053 coverdisk
- **MONROE.PC2** — Marilyn Monroe sketch, from Dali Light (ACM, 1989)
